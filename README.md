# 21-DAY-EXCEL-DATA-ANALYTICS-CHALLENGE
The dataset contains 16 columns and 1000 Rows
The dataset contains freelancer-related information
The table includes details such as freelancer ID, name, gender, age, country, language, primary skill, experience, hourly rate, rating, project count, and client satisfaction.
The dataset has multiple data types
Text data: name, country, primary_skill, Status
Numeric data: age, years_of_experience, rating, no of projects
Decimal/percentage values: client_satisfaction
Mixed categorical values are present in is_active and gender.
There are inconsistencies in categorical columns
The gender column contains different formats such as m, male, MALE, FEMALE, and Female.
The is_active column also contains inconsistent values like 1, 0, TRUE, FALSE, yes, Y, and N.
These values need standardization before analysis.
Some missing or incomplete values are visible
A few cells appear blank in columns like rating, client_satisfaction, and hourly_rate (USD).
Missing values may impact calculations and should be handled during data cleaning.
